Declaration, Initialization, and Accessing Arrays in Java

Arrays in Java are fixed-size, homogeneous data structures that store elements of the same type in contiguous memory locations. They are objects in Java and provide efficient access via indices.

1. Declaration
   - Declaration specifies the type of elements and the array name, but does not allocate memory.
   - Syntax: dataType[] arrayName; or dataType arrayName[];
   - Example: int[] numbers; // Declares an array of integers
             String[] names; // Declares an array of strings

2. Initialization
   - Initialization allocates memory and assigns values to the array.
   - Can be done at declaration time or separately.
   - Syntax for initialization at declaration: dataType[] arrayName = {value1, value2, ...};
   - Syntax for separate initialization: arrayName = new dataType[size];
   - You can also initialize with specific values: arrayName = new dataType[]{value1, value2, ...};
   - Example:
     int[] numbers = {1, 2, 3, 4, 5}; // Initialization with values
     String[] names = new String[3]; // Allocates space for 3 strings, default null
     names[0] = "Alice"; // Assigning values later

3. Accessing Arrays
   - Elements are accessed using zero-based indices (starting from 0).
   - Syntax: arrayName[index]
   - Attempting to access an invalid index throws ArrayIndexOutOfBoundsException.
   - You can use loops (e.g., for, enhanced for) to access all elements.
   - Example:
     int[] numbers = {10, 20, 30};
     System.out.println(numbers[0]); // Outputs 10
     System.out.println(numbers[2]); // Outputs 30
     // Using a loop
     for (int i = 0; i < numbers.length; i++) {
         System.out.println(numbers[i]);
     }
     // Enhanced for loop
     for (int num : numbers) {
         System.out.println(num);
     }

Key Points:
- Arrays have a fixed size once initialized (use ArrayList for dynamic sizes).
- The length property gives the size: arrayName.length
- Multidimensional arrays are arrays of arrays, e.g., int[][] matrix = new int[2][3];

Multidimensional Arrays in Java
Multidimensional arrays are arrays of arrays, allowing storage of data in a tabular form (e.g., matrices). The most common is 2D arrays, but higher dimensions are possible.

1. Declaration
   - Syntax: dataType[][] arrayName; or dataType arrayName[][];
   - For 3D: dataType[][][] arrayName;
   - Example: int[][] matrix; // Declares a 2D array of integers

2. Initialization
   - Can initialize with values: dataType[][] arrayName = {{val1, val2}, {val3, val4}};
   - Or allocate space: arrayName = new dataType[rows][cols];
   - Jagged arrays: Rows can have different lengths, e.g., new int[2][]; then assign each row separately.
   - Example:
     int[][] matrix = {{1, 2, 3}, {4, 5, 6}}; // 2x3 matrix
     int[][] jagged = new int[2][]; // Jagged array
     jagged[0] = new int[3]; // First row has 3 elements
     jagged[1] = new int[2]; // Second row has 2 elements

3. Accessing Elements
   - Use two indices: arrayName[row][col]
   - Zero-based indexing for both dimensions.
   - Use nested loops to traverse.
   - Example:
     int[][] matrix = {{1, 2}, {3, 4}};
     System.out.println(matrix[0][1]); // Outputs 2
     // Traversing with loops
     for (int i = 0; i < matrix.length; i++) { // Rows
         for (int j = 0; j < matrix[i].length; j++) { // Columns
             System.out.print(matrix[i][j] + " ");
         }
         System.out.println();
     }

Key Points for Multidimensional Arrays:
- matrix.length gives the number of rows.
- matrix[i].length gives the number of columns in row i.
- Useful for representing grids, tables, or matrices in applications like games or data processing.
